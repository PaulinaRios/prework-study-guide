Prework Study Guide

HTML

  • HTML stands for HyperText Markup Language.
  • It is used to structure web content.
  • HTML elements are represented by tags.

CSS

  • CSS stands for Cascading Style Sheets.
  • It controls the layout of the web page.
  • CSS can be used for colors, spacing, and font styling.

Git

  • Git is a version control system.
  • It helps track changes in code.
  • GitHub is a cloud-based Git repository hosting service.

JavaScript

  • JavaScript is a programming language for the web.
  • It makes web pages interactive.
  • JavaScript can be used for animations and dynamic content.
